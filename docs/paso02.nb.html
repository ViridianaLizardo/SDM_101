Introducción.

Igual que en el anterior, pero para obtener las capas recortadas en el formato adecuado.

El objetivo de este documento es:

  • Dar un ejemplo reproducible que puedan modificar a su interés.

  • Producir los archivos de capas climáticas que se usarán para modelar una especie con estas características:

    • En formato para Maxent (.asc)

    • Sin multicolinearidad

    • Recortadas al área de entrenamiento

  • Mapa descriptivo de lo anterior

Capas ambientales

La selección de variables ambientales se hace con la función VIF presente en el paquete usdm (Uncertainty and Sensitivity Analysis for Spatial Models). Este paquete proporciona herramientas para el análisis de incertidumbre y sensibilidad en modelos espaciales.

Naimi B, Hamm Na, Groen TA, Skidmore AK, Toxopeus AG (2014). “Where is positional uncertainty a problem for species distribution modelling.” Ecography, 37, 191-203.  doi:10.1111/j.1600-0587.2013.00205.x.

library(usdm)

Recortar capas

En este ejemplo se usarán países para simplificar todo. Se obtiene de los datos pregargados de R llamados World y seleccionaremos los países con registros de D. virginiana.

data('World')
plot(World)
Aviso: plotting the first 9 out of 15 attributes; use max.plot = 15 to plot all

Cargar datos limpios


sf_occ<- list.files('data', 
                    full.names = T, pattern = '.csv') %>% #listar todos los archivos csv
  lapply(., read.csv) %>% # cargarlos
  bind_rows() %>% # concatenarlos
   st_as_sf(coords = c('decimalLongitude', 'decimalLatitude'), # hacer espacial
         crs = 4326)

Búsqueda espacial

Se usa la función st_join que hace una unión espacial y quitamos aquellos polígonos que no fueron unidos a los puntos.

training_area <- st_join(World, sf_occ) %>%
  na.omit() %>%
  tally() # fusionar
plot(training_area)

Cortar las capas raster

env <- crop(wc, training_area, 
     mask = T) 

plot(env)

Reducir variables con VIF

Lo ponemos en una bonita gráfica, cualquier variable mayor a 10 se considera problemática.

vif(env) %>% ggplot(aes(x= Variables, y = VIF)) + 
  geom_col() + coord_cartesian(ylim = c(0,10))

# Excluir con límite de coeficiente de correlación

vif_cor <- vifcor(env,th = 0.75,keep = 'wc2.1_10m_bio_1')
env_cor<- exclude(env, vif_cor) 

plot(env_cor)

# Excluir con pasos

vif_step <- vifstep(env,th = 10,keep = 'wc2.1_10m_bio_1')
env_step<- exclude(env, vif_step) 

plot(env_step)

Guardar el resultado

writeRaster(env_cor,
